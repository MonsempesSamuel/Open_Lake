Management of the lakes
The management of lakes is carried out in the menu lake-list.
An unregistered user can see the entire list of lakes.
For an authenticated user, it is possible to add, modify or delete information on the lake in question.
Management of analyzes
An unregistered user can see all analyzes that match a lake.
The graphs correspond to the average of the data from all analyzes of the selected lake.
For an authenticated user, it is possible to delete / modify an analysis.
The import button allows you to create an analysis.
